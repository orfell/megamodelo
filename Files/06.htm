TPACK
CONTEXTO
PCK

Pedagogía + Contenido

Saber cómo enseñar mejor un contenido específico

El docente transforma el contenido disciplinar en formas pedagógicamente potentes y adaptadas a los estudiantes

Ejemplos: Analogías efectivas, ejemplos contextualizados, secuenciación lógica, estrategias para conceptos difíciles

TCK

Tecnología + Contenido

Usar tecnología para representar o enriquecer el contenido

El docente selecciona tecnologías que mejor representen o transformen la experiencia con el contenido disciplinar

Ejemplos: Simuladores virtuales, visualizadores de datos, geovisualizadores, laboratorios remotos, software disciplinar específico

TPK

Tecnología + Pedagogía

Usar tecnología para apoyar o transformar estrategias de enseñanza

El docente comprende cómo la enseñanza y el aprendizaje cambian cuando se utilizan determinadas tecnologías

Ejemplos: Foros para aprendizaje colaborativo, evaluaciones adaptativas, herramientas de retroalimentación inmediata

TPACK

Integración Completa

Convergencia consciente de los tres saberes en la práctica real

Base operativa del TEdU: docentes capaces de tomar decisiones integradas sobre qué enseñar, cómo enseñarlo y con qué tecnología, de manera cohesiva

Resultado: Experiencias de aprendizaje realmente transformadoras, aprovechando al máximo cada modalidad educativa
